## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

This is a specially crafted autoomated Vulnerability Scanner, Patch Management Tool and Exploitation tool for CVE-2021-44228 / CVE-2021-45046 / CVE-2021-4104. This is a combination of Python, Java and Linux shell program which will make it easier for a Network Administrator, Endpoint Security Developers and Cyber-Security Analysts to perform Penetration Testing and protect themselves from The Log4Shell vulnerability
